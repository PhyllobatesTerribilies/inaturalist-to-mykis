# inaturalist-to-mykis

This program converts the export data from the Mykis community on iNaturalist into a Mykis database-compatible format.

## iNaturalist

[Eine Community für Naturfreunde · iNaturalist](https://www.inaturalist.org/)

iNaturalist is where you can share your findings of animals, plants, and mushrooms with others.

The Mykis Communitiy:
[MykIS-Kartierung · iNaturalist](https://www.inaturalist.org/projects/mykis-kartierung)

## Mykis

Mykis is currently used as a mapping tool for DGFM databases and for offline data collection.

Mykis programm can be ordered per mail under following webadresse:

https://www.tomentella.de/

 **DGFM - Deutsche Gesellschaft für Mykologie e.V.**
 [Verbreitungskarten - www.dgfm-ev.de/de/](https://www.dgfm-ev.de/de/naturschutz/verbreitungskarten)  

## Features

- Converts iNatrualist exports (csv,xlsx) to Mykis comaptible format
- create Mykis file or attach data to Mykis file
- data classification
- data conversion log
